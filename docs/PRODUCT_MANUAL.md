# 贵州扑克 - 产品说明书

## 1. 产品概述

### 1.1 产品名称
**贵州扑克** (Guizhou Poker) - Texas Hold'em Poker Platform

### 1.2 产品定位
一款基于局域网的德州扑克游戏平台，专为朋友聚会、家庭娱乐设计。支持多人实时对战和AI机器人陪玩，纯娱乐性质，不涉及任何真实货币交易。

### 1.3 版本信息
- **当前版本**: v1.5.0
- **发布日期**: 2026-03-21
- **支持平台**: Web (浏览器)
- **网络要求**: 局域网 (LAN)

## 2. 功能特性

### 2.1 核心游戏功能

#### 2.1.1 德州扑克游戏
- **游戏类型**: 德州扑克 (Texas Hold'em)
- **人数支持**: 2-8人
- **游戏流程**:
  1. 翻牌前 (Pre-flop) - 发底牌，大小盲注
  2. 翻牌圈 (Flop) - 发3张公共牌
  3. 转牌圈 (Turn) - 发第4张公共牌
  4. 河牌圈 (River) - 发第5张公共牌
  5. 摊牌 (Showdown) - 比牌决胜负

#### 2.1.2 盲注系统
- 自动轮换大小盲注位置
- 支持多档盲注设置:
  - 小盲 5 / 大盲 10
  - 小盲 10 / 大盲 20
  - 小盲 25 / 大盲 50
  - 小盲 50 / 大盲 100

#### 2.1.3 AI机器人
- 智能AI陪玩，支持2-7个机器人
- AI具有不同难度和策略
- 自动决策：跟注、加注、弃牌、全下

### 2.2 用户系统

#### 2.2.1 账号管理
- **注册登录**: 用户名+密码，支持4-20位用户名，6位以上密码
- **游客登录**: 无需注册，一键快速游戏
- **密码找回**: 暂不支持，建议记住密码

#### 2.2.2 个人资料
- **头像选择**: 20+款精美头像
- **昵称修改**: 随时更改显示名称
- **筹码管理**: 初始1000积分，支持增减记录

#### 2.2.3 用户状态
- 在线状态实时显示
- 游戏中/房间中/离线状态

### 2.3 房间系统

#### 2.3.1 房间管理
- **创建房间**: 自定义房间名、人数、盲注
- **密码保护**: 可选密码保护私密房间
- **房间列表**: 实时显示所有可加入房间
- **房主权限**: 房主可删除房间、管理玩家

#### 2.3.2 游戏控制
- **准备系统**: 玩家准备后开始游戏
- **游戏中退出**: 支持掉线重连（60秒倒计时）
- **游戏结束**: 可选择留在房间继续或返回大厅

### 2.4 社交功能

#### 2.4.1 排行榜
- 全服积分排行榜
- 实时显示玩家状态（在线/游戏中/离线）
- 前三名特殊标识

#### 2.4.2 好友系统 (预留)
- 添加好友
- 查看好友在线状态
- 邀请好友游戏

### 2.5 管理功能

#### 2.5.1 超管权限
- 查看所有用户列表
- 修改用户筹码
- 删除用户账号
- 清空所有房间

## 5. 使用指南

### 5.1 快速开始

#### 5.1.1 启动服务器
```bash
# 进入项目目录
cd Texas-Hold-em-LAN/backend

# 安装依赖（首次）
npm install

# 启动服务器
npm start

# 或使用开发模式
npm run dev
```

#### 5.1.2 访问游戏
1. 确保服务器已启动
2. 在同一局域网的其他设备上打开浏览器
3. 访问 `http://服务器IP:3000`
4. 注册账号或游客登录

### 5.2 游戏流程

#### 5.2.1 创建游戏
1. 登录后进入游戏大厅
2. 点击"创建房间"
3. 设置房间名称、人数、盲注
4. 可选设置密码
5. 点击"创建"

#### 5.2.2 加入游戏
1. 在大厅浏览房间列表
2. 选择状态为"等待中"的房间
3. 点击"加入"
4. 如有密码，输入密码

#### 5.2.3 开始游戏
1. 房主点击"开始游戏"
2. 系统发牌，开始第一轮下注
3. 玩家依次行动：弃牌、跟注、加注、全下
4. 完成所有下注轮后比牌
5. 赢家获得底池

#### 5.2.4 游戏结束
1. 单局结束后显示结果
2. 玩家可选择：
   - "继续游戏" - 留在房间开始下一局
   - "返回大厅" - 离开房间回到大厅

### 5.3 操作说明

#### 5.3.1 游戏内操作
| 操作 | 说明 |
|------|------|
| 弃牌 (Fold) | 放弃当前手牌，不再参与本局 |
| 跟注 (Call) | 跟随当前最高下注额 |
| 加注 (Raise) | 提高下注额 |
| 全下 (All-in) | 押上所有筹码 |
| 过牌 (Check) | 不下注（仅当无人下注时） |

#### 5.3.2 快捷键
- 无快捷键支持，全部鼠标操作

## 6. 游戏规则

### 6.1 德州扑克基本规则

#### 6.1.1 牌型大小（从大到小）
1. **皇家同花顺** (Royal Flush) - 同花色的10-J-Q-K-A
2. **同花顺** (Straight Flush) - 同花色的连续5张
3. **四条** (Four of a Kind) - 四张相同点数
4. **葫芦** (Full House) - 三张+一对
5. **同花** (Flush) - 同花色任意5张
6. **顺子** (Straight) - 连续点数的5张
7. **三条** (Three of a Kind) - 三张相同点数
8. **两对** (Two Pair) - 两个不同的对子
9. **一对** (One Pair) - 一对相同点数
10. **高牌** (High Card) - 单张比大小

#### 6.1.2 下注轮次
1. **Pre-flop**: 发2张底牌后，从大小盲注位置开始下注
2. **Flop**: 发3张公共牌后下注
3. **Turn**: 发第4张公共牌后下注
4. **River**: 发第5张公共牌后下注
5. **Showdown**: 摊牌比大小

### 6.2 特殊情况

#### 6.2.1 全下 (All-in)
- 玩家筹码不足时可全下
- 全下后不再参与后续下注
- 可能产生边池 (Side Pot)

#### 6.2.2 分池 (Split Pot)
- 平局时底池由胜者平分
- 筹码不足时按筹码比例分配

## 7. 常见问题

### 7.1 连接问题
**Q: 无法连接服务器**
- 检查服务器是否已启动
- 确认客户端和服务器在同一局域网
- 检查防火墙设置，确保3000端口开放

**Q: 页面加载缓慢**
- 检查网络连接
- 清除浏览器缓存
- 尝试使用Chrome浏览器

### 7.2 游戏问题
**Q: 游戏卡住不动**
- 刷新页面重连
- 检查是否有玩家掉线
- 联系房主重新开始

**Q: 筹码突然变少**
- 查看筹码流水记录
- 确认是否在游戏中输掉
- 联系管理员查询

### 7.3 账号问题
**Q: 忘记密码**
- 目前不支持密码找回
- 建议重新注册账号

**Q: 如何修改昵称/头像**
- 点击右上角头像进入个人资料
- 在个人信息页面修改
