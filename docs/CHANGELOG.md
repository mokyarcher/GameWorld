## [v1.1.0] - 2026-03-22

### 人机对战系统 (feat)

#### 新增功能
- **人机创建机制**
  - 房主可在等待房间添加人机玩家
  - 支持最多20个不同头像的人机
  - 人机自动准备，无需等待
  - 人机头像路径: `frontend/avatars/bots/bot_1.png` ~ `bot_20.png`

- **人机AI决策系统**
  - 自动决策逻辑: fold/call/check/raise/allin
  - 模拟思考时间: 1-3秒随机延迟
  - 策略: 手牌好则加注，一般则跟注/看牌，差则弃牌
  - 筹码不足时自动全下

- **人机状态同步**
  - 人机玩家状态正确广播给所有客户端
  - 人机标识: `isBot: true`，紫色边框 + 🤖徽章
  - 游戏结算时人机自动继续（筹码≥1000）

#### 技术实现
```javascript
// 后端: backend/games/poker/poker.socket.js
- createBotPlayer(seatNumber, roomId) - 生成人机玩家
- botAction(game, io, player) - 人机AI决策
- processGameDecisions() - 处理人机自动继续逻辑

// 前端: frontend/games/poker/room.html, game.html
- 人机头像显示（紫色边框）
- "添加人机"按钮（仅房主可见）
```

### 游戏流程优化 (opt)

#### 结算离开优化
- **玩家离开机制**
  - 点击"离开房间"后立即跳转到大厅，无需等待
  - 玩家选择离开时立即从数据库删除
  - 防止玩家重新进入已离开的房间

- **房间自动解散**
  - 当房间只剩人机时，自动解散房间
  - 所有真人玩家离开后，不让人机继续对战
  - 发送"房间即将关闭"消息给所有在线玩家

#### 倒计时优化
- 游戏开始倒计时从5秒改为**3秒**
- 游戏结算决策倒计时保持10秒

### UI/UX 优化 (design)

#### 弹窗系统重构
- **自定义弹窗替换原生alert**
  - 深色背景 + 金色/紫色边框
  - 游戏风格统一设计
  - 支持确认/取消按钮
  - 图标提示（👋 人数不足等）

- **人数不足提示优化**
  - 错误弹窗改为友好提示
  - 显示: "👋 人数不足 即将返回大厅"
  - 3秒倒计时后自动跳转

#### 牌桌设计优化
- **牌桌尺寸调整**
  - 宽度: 90% → 95%
  - 高度: 85% → 88%
  - 垂直位置: top 45% → 52%（更靠下）
  - 最大尺寸: 1000x520px → 1100x580px

- **牌桌形状**
  - 椭圆比例: `border-radius: 50% / 40%`（更扁）
  - 新增木纹边框效果
  - 多层阴影: 内阴影 + 外阴影 + 双边框
  - 桌面中心高光效果

#### 座位布局优化
- **6人桌座位位置**
  ```
  seat-0 (中下/本人): bottom -2%  (最底部)
  seat-1 (左下):      bottom 15%, left 5%
  seat-2 (左上):      top 15%, left 5%
  seat-3 (上中):      top -2%    (最顶部)
  seat-4 (右上):      top 15%, right 5%
  seat-5 (右下):      bottom 15%, right 5%
  ```

#### 玩家信息卡优化
- **信息卡样式**
  - 深色渐变背景 + 毛玻璃效果
  - 边框: 1px solid rgba(255,255,255,0.1)
  - 圆角: 12px
  - 阴影: 0 4px 20px rgba(0,0,0,0.5)

- **头像显示**
  - 尺寸: 60px（适中）
  - 边框: 3px solid rgba(255,255,255,0.3)
  - 激活状态: 绿色边框 + 脉冲动画

- **位置标记**
  - BTN (庄家): 蓝色渐变
  - SB (小盲): 橙色渐变
  - BB (大盲): 红色渐变

#### 手牌显示优化
- **本人手牌（中下位置）**
  - 方向: 横向铺开（左右叠放）
  - 位置: 信息框上方
  - 尺寸: 50x70px
  - 重叠: -20px 负边距节省空间
  - 与信息框间距: 45px

- **其他玩家手牌**
  - 默认不显示（保护隐私）
  - 游戏结束时显示

#### 公共牌优化
- **尺寸缩小**
  - 宽度: 60px → 50px
  - 高度: 84px → 70px
  - 字体: 相应缩小

- **位置调整**
  - 垂直位置: top 48% → 44%
  - 与底池间距更合理

#### 底池显示优化
- **移除顶部底池显示**
  - 顶部信息栏只保留: 房间名、盲注、退出按钮
  - 底池只在桌面中央显示

- **底池框样式**
  - 位置: top 26%
  - 尺寸: 更小巧（padding: 10px 25px）
  - 圆角: 20px
  - 边框: 1px
  - 筹码图标: 18px

#### 倒计时位置优化
- **倒计时显示**
  - 位置: 头像上方（长条样式）
  - 背景: 金色渐变
  - 文字: 白色/黑色
  - 动画: 脉冲效果

#### 徽章位置调整
- **断线徽章**: 头像左下角
- **人机徽章**: 头像右下角
- **倒计时**: 头像上方（避免重叠）
